         🚀 КАК ЗАГРУЗИТЬ ПРОЕКТ НА GITHUB
═══════════════════════════════════════════════════════════════

⚠️ ПРОБЛЕМА: Слишком много файлов?

РЕШЕНИЕ: Git автоматически ИСКЛЮЧИТ ненужные файлы!
         (node_modules, tmp, архивы и т.д.)

───────────────────────────────────────────────────────────────
✅ БЫСТРОЕ РЕШЕНИЕ (копируй-вставь)
───────────────────────────────────────────────────────────────

# 1. Подготовка (выполните один раз)
cd /home/runner/app
./prepare-for-github.sh

# 2. Инициализация Git
git init
git add .
git commit -m "Initial commit: Chat app with file upload"

# 3. Создайте репозиторий на GitHub
Откройте: https://github.com/new
Название: my-chat-app (или любое)
Тип: Public или Private

# 4. Загрузите код (ЗАМЕНИТЕ URL!)
git remote add origin https://github.com/ваш-username/ваш-репо.git
git branch -M main
git push -u origin main

───────────────────────────────────────────────────────────────
📊 ЧТО БУДЕТ ЗАГРУЖЕНО
───────────────────────────────────────────────────────────────

✅ БУДЕТ (~5-10 MB):
   ├─ app/              (код приложения)
   ├─ config/           (конфигурация)
   ├─ db/               (миграции)
   ├─ lib/              (генераторы)
   ├─ spec/             (тесты)
   ├─ Dockerfile        (для деплоя)
   ├─ Gemfile           (Ruby зависимости)
   ├─ package.json      (JS зависимости)
   └─ README файлы      (документация)

❌ НЕ БУДЕТ (автоматически):
   ├─ node_modules/     (~400 MB) - установится через npm
   ├─ tmp/              (временные файлы)
   ├─ log/              (логи)
   ├─ storage/          (загруженные файлы)
   └─ *.tar.gz          (архивы)

───────────────────────────────────────────────────────────────
🔧 ЕСЛИ ВОЗНИКЛИ ПРОБЛЕМЫ
───────────────────────────────────────────────────────────────

ПРОБЛЕМА: "File too large"
РЕШЕНИЕ:
  git rm --cached большой-файл
  git commit -m "Remove large file"
  git push

ПРОБЛЕМА: "Permission denied"
РЕШЕНИЕ:
  # Используйте Personal Access Token
  https://github.com/settings/tokens
  
ПРОБЛЕМА: Долго загружается
РЕШЕНИЕ:
  # Это нормально! Подождите 2-5 минут

───────────────────────────────────────────────────────────────
📱 АЛЬТЕРНАТИВА: GitHub Desktop
───────────────────────────────────────────────────────────────

Если командная строка не работает:

3. Нажмите "Publish repository"
4. Готово!
