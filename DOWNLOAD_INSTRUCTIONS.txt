🚀 ДЕПЛОЙ НА RENDER.COM (3 ШАГА)
───────────────────────────────────────────────────────────────

ШАГ 1: Загрузите на GitHub
$ cd app
$ git init
$ git add .
$ git commit -m "Chat app with file upload"
$ git push origin main

ШАГ 2: Создайте сервисы на Render.com
- PostgreSQL (БД)
- Web Service (Приложение)

ШАГ 3: Настройте переменные окружения
DATABASE_URL=...
SECRET_KEY_BASE=...
LLM_API_KEY=...

⏱️ ВРЕМЯ: ~5 минут
📖 ПОДРОБНОСТИ: README_DEPLOYMENT.md
